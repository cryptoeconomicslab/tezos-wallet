# Tezos Wallet

## Development

### How to install

```bash
$ cd app/
$ npm i
```

### How to run

```bash
$ npm run android
```

### eslint

```bash
$ npm run lint
$ npm run lint:fix
```

### prettier

```bash
$ npm run fmt
```
