# Tezos Wallet

## Development

install package
```sh
$ cd app/
$ npm i 
```

start
```sh
$ npm start
```

eslint
```sh
$ npm run elc // eslint
$ npm run slf // eslint --fix
```

prettier
```sh
$npm run fmt // prettier

```
